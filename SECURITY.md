# Security Policy

## Supported Versions

Security fixes are applied to the latest commit on `main`.

## Reporting A Vulnerability

Please do not report security issues in public GitHub issues.

Use one of these private channels:

1. GitHub Security Advisories (preferred):
   - Open a private report from the repository Security tab.
2. Contact the repository owner directly on GitHub:
   - [https://github.com/zhuisdev](https://github.com/zhuisdev)

Include:

- affected file(s) and behavior
- steps to reproduce
- impact assessment
- any suggested fix

## Response Process

- Initial acknowledgement target: within 7 days.
- After validation, a fix will be prepared and released on `main`.
